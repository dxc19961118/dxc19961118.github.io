如何使用

PM Skills 是 AI Agent 的产品经理技能包，适配多种工具

🤖 Claude Cowork（推荐非开发者）

  1. 点击左下角 Customize
  2. 进入 Browse plugins → Personal → +
  3. 选择 Add marketplace from GitHub
  4. 输入 phuryn/pm-skills

9 个插件全部自动安装，命令和技能即刻可用。

️ Claude Code（CLI）

  1. 添加市场源：claude plugin marketplace add phuryn/pm-skills
  2. 按需安装插件：claude plugin install pm-toolkit@pm-skills

可逐个安装，也可一次安装全部 9 个插件。安装后在对话中使用 /命令名 即可触发工作流。

🧠 Codex CLI（OpenAI）

  1. 添加市场源：codex plugin marketplace add phuryn/pm-skills
  2. 安装插件：codex plugin add pm-toolkit@pm-skills

Codex 读取相同的市场文件，无需转换。但 / 斜杠命令不会作为 Codex 命令运行——用自然语言描述步骤即可触发相同工作流。

🔌 其他 AI 助手（仅技能）

SKILL.md 文件遵循通用技能格式，支持：

  • Gemini CLI → 复制到 .gemini/skills/
  • Cursor → 复制到 .cursor/skills/
  • OpenCode → 复制到 .opencode/skills/
  • Kiro → 复制到 .kiro/skills/

斜杠命令是 Claude 特有的，其他工具仅支持技能。

⚙️ 工作原理

技能（Skills）：每个技能编码一个经过验证的 PM 框架（发现、假设映射、优先级排序、策略等），提供领域知识和分析框架。

命令（Commands）：用 /命令名 触发的端到端工作流，串联一个或多个技能。例如 /discover 串联了 4 个技能。

插件（Plugins）：将相关技能和命令打包成可安装单元，每个插件覆盖一个 PM 领域。

💡 技能自动加载

技能会在对话相关时自动加载，无需手动调用。

如需强制加载（例如让技能优先于通用知识），可使用：

  • /插件名:技能名 — 精确加载
  • /技能名 — Claude 会自动补全前缀
